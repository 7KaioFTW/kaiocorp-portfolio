# playground/

Prototypes HTML autonomes et **jetables** (Rulebook R6.12) : on itère visuellement, on garde le
prompt/skill qui les génère — pas le HTML comme source de vérité.

Esquisser en HTML même quand la cible est React (R10.5). Terminer les artefacts interactifs par un
export « copier en JSON / en prompt » (R10.8).

Contenu actuel :
- `prototype-homepage.html` — miroir visuel autonome de la homepage (ouvrir dans un navigateur).
  La vraie implémentation vit dans `src/` ; ce fichier sert d'aperçu rapide sans lancer Next.
